This project is a simple Udemy homepage clone built using HTML and CSS. It replicates the core layout and design structure of the original Udemy website, including interactive hover effects and modern course card styling.

The goal of this project is to practice layout design, Flexbox positioning, responsive structure, and UI styling techniques.

**1. Navigation Bar**

- Logo section
- Search bar
- Course links
- Icons for cart, notifications, and profile
- Clean horizontal alignment

**2. Categories Section**

- Scrollable / aligned category buttons
- Styled pill-shaped category badges
- Hover effects for interactivity

**3. Sale Banner Section**

- Large promotional banner
- Eye-catching hero image
- Clean spacing and alignment

**4. Recommended Courses**

- Course cards with: Thumbnail image
- Course title
- Instructor name
- Rating with stars
- Price display
- Hover effect on course cards

**5. Topics to Discover**

- Styled topic boxes
- Structured layout for easy browsing

**6. Most Popular Courses**

- Grid layout of multiple course cards
- Consistent spacing
- Responsive alignment
- Rating and pricing UI

**7. Footer**

- Multi-column footer layout
- Important links (Business, Careers, Blog, etc.)
- Clean dark theme design

**Technologies Used**

HTML5, 
CSS3, 
Flexbox, 
Hover Effects.

**Learning Outcomes**

- Improved understanding of layout structuring
- Mastered Flexbox and Grid for card layouts
- Practiced UI spacing and alignment
- Implemented hover interactions
- Designed a real-world homepage clone

**Project Purpose**
This project was created for learning and practice purposes to strengthen front-end development skills by recreating a real-world website layout.
